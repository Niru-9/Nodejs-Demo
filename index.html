Running JavaScript with Node.js

Learn how to execute JavaScript code outside the browser

What is Node.js?

Node.js is a JavaScript runtime built on Chrome's V8 JavaScript engine. It allows you to run JavaScript code on your computer as a standalone program, not just in a browser.

Key Advantages of Node.js

  • Allows JavaScript to run on the server-side
  • Provides access to the file system and operating system
  • Has a huge ecosystem of packages via npm
  • Excellent for building web servers and CLI tools

How to Install Node.js

  1. Visit the official Node.js website at nodejs.org
  2. Download the LTS (Long Term Support) version for your operating system
  3. Run the installer and follow the installation instructions
  4. Verify the installation by opening a terminal and typing: 
    node --version

Node.js vs Browser JavaScript

In Node.js you can:

  • Access the file system
  • Create web servers
  • Run shell commands
  • Use environment variables

In the browser you can:

  • Manipulate the DOM
  • Interact with user events
  • Use browser APIs
  • Make HTTP requests
